# 指令集

## 1. 基础分类处理 (classifier.batch_process)

```sh
uv run python -m classifier.batch_process --cache .\cache\lastframe_256 --output-root .\output\final\diff_ths\th80 --period-threshold 35.0 --sign-threshold 0.8 --ignore-last-frames 3 --direct
```

**参数说明：**
- `--cache` - 输入缓存目录路径
- `--output-root` - 输出根目录路径
- `--period-threshold` - 周期阈值 (默认: 6.0)
- `--sign-threshold` - 信号率阈值 (0.0-1.0, 默认: 0.9)
- `--ignore-last-frames` - 忽略最后N帧以除去自注意力影响 (默认: 3)
- `--direct` - 跳过图像生成，仅输出labels.csv文件

**输出：** 生成per-run的labels.csv和JSON结果文件

---

## 2. 标签聚合 (ablation.aggregate_labels)

```sh
uv run python -m ablation.aggregate_labels --csv_dir .\output\final\diff_ths\th80 --output .\output\label\ --name aggregated_labels.csv
```

**参数说明：**
- `--csv_dir` - 包含labels.csv文件的目录
- `--output` - 输出目录路径
- `--name` - 输出文件名 (默认: aggregated_labels.csv)

**处理逻辑：** 对目录下所有labels.csv文件进行多数表决，处理不同大小的矩阵

**标签含义：**
- `-1` - Wave Head
- `1` - Anchor Head
- `2` - Veil Head
- `0` - Unknown

---

## 3. 批处理 - 变参sign-threshold (batch/ths.py)

```sh
uv run batch/ths.py --cache .\cache\diff_prompts\lastframe_256 --output-root .\output\period6\diff_ths --agg-output .\output\csv\final2.0\ths --period 6.5 --ths 70 80 90 --workers 3 --compare
```

**参数说明：**
- `--cache` - 输入缓存目录
- `--output-root` - 输出根目录
- `--agg-output` - 聚合输出目录
- `--period` - 周期阈值
- `--ths` - sign-threshold值列表，以空格分隔 (默认: 70 80 90)
- `--workers` - 并行工作线程数
- `--compare` - 执行对比步骤 (可选)
- `--name-template` - 输出文件名模板 (可选)
- `--origin` - 原始CSV路径用于对比 (可选)
- `--ignore-last-frames` - 忽略最后N帧 (默认: 3)

**工作流：** 对每个ths值，依次执行classifier.batch_process和aggregate_labels

---

## 4. 批处理 - 变参period-threshold (batch/thp.py)

```sh
uv run batch/thp.py --cache .\cache\diff_prompts\lastframe_256 --output-root .\output\period6\diff_thp --agg-output .\output\csv\final2.0\thp --thps 6.5 7.0 8.0 --ths 80 --workers 3 --compare
```

**参数说明：**
- `--cache` - 输入缓存目录
- `--output-root` - 输出根目录
- `--agg-output` - 聚合输出目录
- `--thps` - period-threshold值列表，以空格分隔
- `--ths` - sign-threshold值 (单个，默认: 80)
- `--workers` - 并行工作线程数
- `--compare` - 执行对比步骤 (可选)
- `--sign` - 指定sign-threshold (可选)
- `--ignore-last-frames` - 忽略最后N帧 (默认: 3)

**工作流：** 对每个thp值，结合ths值执行完整管道

**输出目录格式：** `thp{thp}_th{ths}/` (如 thp6p5_th80)

---

## 5. 多缓存批处理 (batch/multi_cache.py)

```sh
uv run batch/multi_cache.py --cache-root .\cache\diff_frame --output-root .\output\period10\diff_frame --agg-output .\output\csv\final2.0\multi --workers 4
```

**参数说明：**
- `--cache-root` - 缓存根目录 (遍历其子目录)
- `--output-root` - 输出根目录
- `--agg-output` - 聚合输出目录
- `--workers` - 并行工作线程数
- `--period` - 周期阈值 (默认: 10.0)
- `--ths` - sign-threshold值 (默认: 80)
- `--sign-threshold` - sign-threshold值(分数制) (默认: 0.8)
- `--compare` - 执行对比步骤 (可选)
- `--dry-run` - 模拟运行，不执行命令 (可选)

**工作流：** 遍历cache-root下所有子目录，对每个执行complete pipeline

**内存监测：** 使用psutil采样内存/CPU使用情况(可选)

---

## 6. CSV对比热力图 (grid_graph/compare.py)

```sh
uv run python grid_graph/compare.py --origin .\output\label\origin.csv --target .\output\label\target.csv --output .\output\pic\compare.png
```

**参数说明：**
- `--origin` - 原始CSV文件路径
- `--target` - 目标CSV文件路径
- `--output` - 输出PNG图像路径

**差异显示：** 相同值显示灰色，不同值显示Origin颜色

---

## 7. 单CSV热力图 (grid_graph/plot_single.py)

```sh
uv run python grid_graph/plot_single.py --csv .\output\label\aggregated_labels.csv --output .\output\pic\heatmap.png --transpose
```

**参数说明：**
- `--csv` - 输入CSV文件路径
- `--output` - 输出PNG图像路径
- `--transpose` - 转置矩阵 (可选)

**输出：** 带图例的单热力图，标注各值含义

**标签颜色映射：**
- `#86e3ce` - Wave Head (-1)
- `#ffd1aa` - Anchor Head (1)
- `#bab3fa` - Veil Head (2)
